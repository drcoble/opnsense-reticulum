# Phase 3 — Jinja2 Config Templates

## Overview
Jinja2 templates transform OPNsense config.xml data into native daemon config files. OPNsense's configd renders these via `configctl template reload OPNsense/Reticulum`.

---

## Files

| Template | Output Path | Purpose |
|----------|------------|---------|
| `+TARGETS` | (mapping file) | Maps templates to output paths |
| `reticulum_config.j2` | `/usr/local/etc/reticulum/config` | rnsd INI config |
| `lxmf_config.j2` | `/usr/local/etc/lxmf/config` | lxmd INI config |
| `rc.conf.d_rnsd.j2` | `/etc/rc.conf.d/rnsd` | rnsd boot enable |
| `rc.conf.d_lxmd.j2` | `/etc/rc.conf.d/lxmd` | lxmd boot enable |
| `lxmf_allowed.j2` | `/usr/local/etc/lxmf/allowed` | Identity whitelist |
| `lxmf_ignored.j2` | `/usr/local/etc/lxmf/ignored` | Destination blacklist |

All templates live at: `src/opnsense/service/templates/OPNsense/Reticulum/`

## 2. reticulum_config.j2 — Reticulum INI Config

### Template Variable Access
OPNsense model data is accessed via `OPNsense.Reticulum.<container>.<field>`. For ArrayField items, iterate with a for loop.

### Boolean Mapping
OPNsense stores booleans as `"1"` / `"0"`. Reticulum expects `True` / `False` or `Yes` / `No`.

Helper macro:
```jinja2
{% macro bool(val) %}{% if val == '1' %}True{% else %}False{% endif %}{% endmacro %}
{% macro yesno(val) %}{% if val == '1' %}yes{% else %}no{% endif %}{% endmacro %}
```

## 6. Edge Cases

| Case | Handling |
|------|---------|
| First install, no data in config.xml | All `default()` filters produce valid minimal config |
| No interfaces defined | `[reticulum]` and `[logging]` sections only; no `[[...]]` sections |
| All interfaces disabled | Same as no interfaces (loop skips disabled) |
| Empty optional fields | Omitted from output via `!= ''` checks |
| RNodeMultiInterface raw block | Emitted verbatim with no indentation processing |
| CSV list with single item | Renders as single value (valid for Reticulum) |
| lxmd with propagation disabled | `[propagation]` section emitted with `enable_node = no` only |
